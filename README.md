## Description

Work-in-progress to opensource stacks on chain.
Nestjs app that connects to the stacks blockchain DB. [read more here](https://docs.hiro.so/get-started/running-api-node)

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
